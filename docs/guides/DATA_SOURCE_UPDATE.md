# 数据源更新说明

## 变更概述

已将ETF量化分析系统的数据源从东方财富(东财)切换至新浪财经和同花顺。

## 新数据源

### 1. 实时净值行情
- **数据源**: 同花顺
- **接口**: `ak.fund_etf_spot_ths()`
- **数据内容**: 基金净值数据（非实时市场价格）
  - 当前单位净值
  - 增长率（涨跌幅）
  - 增长值（涨跌额）
- **注意事项**: 
  - 提供的是基金净值数据，非实时市场成交价格
  - 增长率可以反映基金的涨跌情况
  - 适合用于选股和策略分析

### 2. ETF列表
- **数据源**: 新浪财经
- **接口**: `ak.fund_etf_category_sina(symbol='ETF基金')`
- **数据内容**:
  - ETF代码
  - ETF名称
  - 实时价格
  - 涨跌幅

### 3. 历史行情数据
- **数据源**: 新浪财经
- **接口**: `ak.fund_etf_hist_sina(symbol='sh510300')`
- **数据内容**:
  - 日期
  - 开盘价
  - 最高价
  - 最低价
  - 收盘价
  - 成交量
  - 成交额
- **支持**: 支持前复权、后复权、不复权

### 4. ETF分红信息
- **数据源**: 新浪财经
- **接口**: `ak.fund_etf_dividend_sina(symbol='sh510300')`

## 数据源对比

| 功能 | 旧数据源(东财) | 新数据源 | 说明 |
|------|---------------|---------|------|
| 实时行情 | 东方财富 | 同花顺 | 基金净值数据 vs 实时价格 |
| ETF列表 | 东方财富 | 新浪财经 | 同等数据 |
| 历史数据 | 东方财富 | 新浪财经 | 同等数据 |
| 分红信息 | - | 新浪财经 | 新增功能 |

## 重要说明

### 实时净值 vs 实时价格

**基金净值**:
- 每日公布一次
- 反映基金的单位价值
- 适合长期投资分析
- 增长率可作为涨跌参考

**实时价格**:
- 交易时间内实时更新
- 反映市场成交价格
- 适合短线交易

本系统现在使用**基金净值**数据进行实时查询：
- 基金净值每日更新
- 增长率反映基金表现
- 适用于策略选股和趋势分析

### 适用场景

✅ **适合使用本系统的场景**:
- 基于技术指标的选股
- MACD、布林带等策略分析
- 历史数据回测
- 长期投资分析
- 基金净值走势分析

⚠️ **需要注意的场景**:
- 超短线交易（分钟级）
- 实时盘口数据
- 买卖一价、委比等

## 使用示例

### 查询实时净值
```python
from data.etf_data_fetcher import ETFDataFetcher

fetcher = ETFDataFetcher()

# 获取所有ETF实时净值
realtime = fetcher.get_etf_realtime()
print(realtime.head())

# 获取涨幅前10名
top_gainers = fetcher.get_top_gainers(10)
print(top_gainers)

# 查询特定ETF
etf_300 = fetcher.get_etf_realtime('510300')
print(etf_300)
```

### 获取历史数据
```python
# 获取历史行情
history = fetcher.get_etf_history('510300', '20240101', '20240226')
print(history)
```
