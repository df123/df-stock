# 策略筛选图表功能

## 功能概述

策略筛选页面添加了ETF技术分析图表功能。点击筛选结果中的任意一行，会弹出对话框显示该ETF的价格走势、布林带和MACD技术指标图表。

## 功能特性

### 1. 点击查看图表
- 在筛选结果表格中，点击任意一行即可查看该ETF的技术分析图表
- 对话框标题显示ETF代码和名称
- 自动加载最近3个月的历史数据

### 2. 主图（价格+布林带）
- 显示ETF收盘价走势（蓝色实线）
- 显示布林带上轨（红色虚线）
- 显示布林带中轨（绿色实线）
- 显示布林带下轨（红色虚线）
- 支持鼠标悬停查看具体数值
- 支持缩放和平移查看历史数据

### 3. MACD图表
- 显示MACD快线（蓝色实线）
- 显示MACD信号线（黄色实线）
- 显示MACD柱状图（正数为绿色，负数为红色）
- 支持鼠标悬停查看具体数值
- 支持缩放和平移查看历史数据

### 4. 交互功能
- **缩放**：鼠标滚轮缩放，或拖动底部滑块
- **平移**：在图表区域按住鼠标拖动
- **悬停**：鼠标悬停显示详细数据
- **图例**：点击图例可显示/隐藏对应线条

## 图表说明

### 布林带（Bollinger Bands）
- **上轨**：中轨 + 2倍标准差
- **中轨**：20日移动平均线
- **下轨**：中轨 - 2倍标准差
- 价格突破上轨表示超买信号
- 价格跌破下轨表示超卖信号

### MACD（指数平滑异同移动平均线）
- **MACD快线**：DIF = 12日EMA - 26日EMA
- **信号线**：DEA = DIF的9日EMA
- **柱状图**：MACD = 2 × (DIF - DEA)
- 金叉：快线上穿信号线，看涨信号
- 死叉：快线下穿信号线，看跌信号

## 使用方法

1. 执行策略筛选
2. 在筛选结果表格中，点击感兴趣的ETF行
3. 查看弹出的技术分析图表
4. 使用鼠标交互功能查看详细信息
5. 点击关闭按钮或ESC键关闭对话框

## 技术实现

- 使用 ECharts 5.4.0 绘制图表
- 数据来源：`GET /api/history/{code}/indicators`
- 数据周期：最近3个月
- 响应式设计：支持窗口大小变化
- 内存管理：对话框关闭时自动销毁图表实例
